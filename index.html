Tips!

  • To start the game you can use buttons on the bottom control panel
  • Click start button or Enter key in your keyboard to start a new game
  • Click pause button or Enter after the game starts to pause the game
  • Click restart button or R key in your keyboard to reset the game
  • You can only control your ping pong pad with mouse, move up and down to move the pad in mouses direction
  • Once you started the game you will play against computer AI
  • Every time the ball hits the pads pad's length will be cut and the ball gets faster speed, the game will go harder every second
  • The game is very simple, thank you for playing!
  • Have fun!
